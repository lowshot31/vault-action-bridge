# 보안 정책

Vault Pilot는 선택된 노트 내용을 읽고, 설정된 모델 프로바이더를 호출하며, 사용자 볼트의 파일에 검토된 변경 사항을 적용할 수 있는 Obsidian 데스크톱 플러그인입니다.

## 지원 버전

최신 GitHub 릴리스만 보안 수정을 지원합니다.

## 취약점 보고

개인 데이터를 노출하지 않는 문제는 GitHub Issues를 통해 보고해 주세요. 민감한 세부 사항이 포함된 문제는 공개 재현 절차를 게시하기 전에 메인테이너에게 비공개로 연락해 주세요.

보고 시 포함할 사항:

- 플러그인 버전
- Obsidian 버전
- 운영 체제
- 사용한 프로바이더 모드 (`openai-oauth`, API 키 프로바이더, 또는 웹뷰 전용)
- 재현 절차
- 예상 동작
- 실제 동작
- 볼트 파일이나 자격 증명이 노출되었을 가능성 여부

## 보안 모델

이 플러그인은 AI 쓰기를 명시적인 검토 단계 뒤에 둡니다:

1. 모델이 볼트 액션 JSON을 제안할 수 있습니다.
2. 플러그인이 요청된 액션과 볼트 상대 경로를 검증합니다.
3. 사용자가 액션 요약과 JSON을 검토합니다.
4. 플러그인은 확인 후에만 변경을 적용합니다.

절대 경로와 `..` 경로 탐색은 거부됩니다.

## 네트워크 사용

네트워크 동작은 사용자 설정에 따라 달라집니다:

- 현재 노트 및 선택 텍스트 질문은 설정된 프로바이더로 전송됩니다.
- `openai-oauth` 모드는 설정된 로컬 프록시 URL(일반적으로 `http://127.0.0.1:10531/v1`)로 요청을 전송합니다.
- API 키 프로바이더 모드는 선택한 프로바이더 엔드포인트로 요청을 전송합니다.

## 자격 증명 처리

API 키와 플러그인 설정은 Obsidian의 `loadData()` 및 `saveData()` API를 사용하여 Obsidian 플러그인 데이터에 저장됩니다. 선택한 프로바이더가 수신할 수 없는 노트나 자격 증명과 함께 이 플러그인을 사용하지 마세요.

## 도구 설치

플러그인에는 Node.js, Codex, `openai-oauth`를 위한 설정 버튼이 포함되어 있습니다. 이 버튼들은 사용자가 누른 후 표시되는 터미널에서 명령을 실행합니다. 플러그인은 프로그램을 자동으로 설치하거나 백그라운드에서 인증하지 않아야 합니다.

## 텔레메트리

Vault Pilot에는 클라이언트 측 텔레메트리나 분석 기능이 포함되어 있지 않습니다.
